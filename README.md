# Supermarket-Analysis
# Supermarket Sales Analysis

## Project Overview

This project analyzes supermarket sales data using **Python** for data cleaning and exploratory data analysis (EDA), and **Power BI** for building interactive dashboards. The goal is to provide valuable business insights into sales performance, customer behavior, product profitability, and branch performance to support data-driven decision-making.

---

# Dashboard Pages

## Executive Overview

This page provides a high-level summary of the supermarket's overall performance. It includes key performance indicators (KPIs) such as Total Sales, Gross Income, Total Transactions, and Average Rating, along with sales trends to help management quickly assess business performance.

---

## Product Performance

This page focuses on product line analysis by comparing sales, revenue, and profitability across different product categories. It helps identify the best-selling products, top-performing categories, and opportunities to improve product performance.

---

## Customer Analysis

This dashboard analyzes customer demographics and purchasing behavior. It includes insights into customer type, gender distribution, payment methods, customer ratings, and buying patterns to better understand customer preferences.

---

## Time & Branch Analysis

This page examines sales performance across different branches and time periods. It highlights sales trends by month, day, and hour while comparing branch performance to identify peak business periods and operational opportunities.
